# cloud-file-manager

Cloud file manager for uploading, organizing, and sharing files with a React/Vite frontend and backend services.

## Features

- File upload and organization workflow.
- File sharing foundation.
- React/Vite user interface.
- Supabase-oriented backend integration.

## Tech Stack

- React
- Vite
- TypeScript
- Tailwind CSS
- Supabase

## Getting Started

```bash
npm install
npm run dev
```

Active cloud file management project.
